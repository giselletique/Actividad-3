Recuerde los Conceptos Basicos de Bases de Datos

Dato: Palabra o número que se utilia para construir información.
Base de Datos: Colección de datos relacionados
Campo: Área de almacenamiento en la base de datos para almacenar datos de un tipo especifico
Registro: Coleccion de Datos iguales o de diferentes tipos o relacionados
Archivo: Coleccion de regitros relacionados
